Welcome to the FDIC Bank Data API documentation. This API provides access to comprehensive banking industry data including financial institution information, historical data, bank failures, and more.

## API Overview

The FDIC Bank Data API (Beta) serves banking industry data to the public, offering access to:

- Financial institution demographic and location information
- Historical data from 1934 onward
- Bank failure records
- Financial information for FDIC insured institutions
- Summary of deposits data
- Demographics information

## Base URL

All API requests should be made to:

```
https://api.fdic.gov/banks
```
